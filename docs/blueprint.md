# **App Name**: AI-GRID Summit

## Core Features:

- Event Information Display: Display event details including title, tagline, dates, venue, and a countdown timer to the event.
- Homepage Highlights: Display key event highlights and quick links to important sections like Schedule, Speakers, Projects, and Contact.
- Schedule Page: Present a fully detailed event schedule for each day (January 6, 7, 8) with timings, talk topics, speakers, and mode (Hybrid/Online).
- Speakers Page: Showcase speaker profiles with photos, affiliations, and talk topics, indicating whether they are presenting Offline, Hybrid, or Online.
- Student Project Demo Details: Outline the format, eligible areas, and certificate provisions for student project demonstrations, specifying timing and mode.
- Registration Integration: Implement a registration page using a Microsoft Form, embedding the provided link for user registration.
- Venue and Contact Information: Provide details about the event venue (Vardhaman College of Engineering) with an embedded Google Maps and include contact information for the organizers.

## Style Guidelines:

- Primary color: Deep blue (#1A5276) to evoke intelligence, innovation, and reliability, fitting the AI and energy theme.
- Background color: Light blue-gray (#D0D3D4), a desaturated version of the primary color, for a clean and professional look.
- Accent color: Vibrant orange (#E67E22) for the call to action buttons to bring some additional life and vibrancy
- Body font: 'Inter', a grotesque-style sans-serif font that lends a machined, objective, neutral look and is suitable for body text
- Headline font: 'Space Grotesk', a proportional sans-serif font, provides a techy and scientific feel that matches AI and smart grid themes.
- Use modern and minimalist icons to represent different event sections and interactive elements.
- Maintain a clean, grid-based layout with ample white space for readability and easy navigation.
- Incorporate smooth transitions between sections and pages to enhance the user experience. Use subtle parallax scrolling effects to add depth and a futuristic feel.
- Implement animated loading states and progress bars for asynchronous content loading. Use animated icons and illustrations where appropriate to add visual interest and engagement.
